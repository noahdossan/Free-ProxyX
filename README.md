## Termux Installation

* `pkg install git`
* `git clone https://github.com/noahdossan/Free-Proxy`
* `cd Free-Proxy`
* `bash Setup.sh`

---
## Installing for linux

* `sudo apt install git`
* `git clone https://github.com/noahdossan/Free-Proxy` 
* `cd Free-Proxy`
* `sudo bash SetupLinux.sh`

---
## Start Proxy Termux and Linux

#### Enter a command to start the proxy.

* `bash FreeProxy.sh`
